# Adding default values

You can easily add default values to your rows by using both `createRow` and `duplicateRow` props.

Try adding rows, inserting rows (using `Shift+Enter`), or duplicating empty rows (using `Ctrl+D`):
<DefaultValues />

```tsx
<DataSheetGrid
  createRow={() => ({ int: 0, date: new Date() })}
  duplicateRow={({ rowData }) => ({
    ...rowData,
    int: rowData.int ?? 0,
    date: rowData.date ?? new Date(),
  })}
/>
```
