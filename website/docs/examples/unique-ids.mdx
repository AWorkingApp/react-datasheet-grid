# Having unique ids

Having a unique id on each row can be achieved with `createRow` and `duplicateRow` props.

Try deleting rows, creating new rows, or duplicating rows:

<UniqueIds />

```tsx
<DataSheetGrid
  createRow={() => ({ id: genId() })}
  duplicateRow={({ rowData }) => ({ ...rowData, id: genId() })}
  columns={[
    {
      ...keyColumn('id', textColumn),
      title: 'ID',
      disabled: true,
      isCellEmpty: () => true
    },
    {
      ...keyColumn('firstName', textColumn),
      title: 'First name'
    },
    {
      ...keyColumn('lastName', textColumn),
      title: 'Last name'
    },
  ]}
/>
```
